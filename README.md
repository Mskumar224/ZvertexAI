# Job Auto-Apply

A web application to automate job applications by allowing users to upload resumes, select target companies, and auto-apply to jobs. The UI is inspired by Perplexity, featuring a dark theme, sidebar navigation, and a clean, modern layout.

## Project Overview

- **Backend**: Node.js with Express, MongoDB, Nodemailer, Multer for file uploads, and JWT for authentication.
- **Frontend**: React with TypeScript, Material-UI (MUI), and React Router for navigation.
- **Features**:
  - User signup/login with subscription types (Student, Vendor, Business).
  - Resume upload (limit of 5 per user, prompts for subscription upgrade).
  - Company selection and auto-apply functionality.
  - Dashboard to track applications.
  - Perplexity-inspired UI with sidebar, search bar, and footer.
